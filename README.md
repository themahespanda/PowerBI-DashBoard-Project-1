# Data Jobs Dashboard with Power Bi

## Project Overview
This dashboard was created for **Job Seekers, Job
Transitioners, and Job Swappers** to solve a common
problem: information about the data job market is
scattered and hard to grasp.Using a **real-world dataset
of 2024 data science job postings (including titles,
salaries, and locations)**, this project provides a
single, easy-to-use interface to explore market trends
and compensation.



##  Why I Built This ?

As job seekers and career transitioners explore opportunities in data and analytics, one challenge persists:

**Job market insights are fragmented, inconsistent, and difficult to interpret.**

This project transforms raw 2024 data science job postings into a centralized, interactive dashboard that empowers professionals to:

- Benchmark salaries
- Identify high-demand roles
- Compare geographic hiring trends
- Make data-driven career decisions

---

##  What This Dashboard Delivers

✔ Median salary insights by role  
✔ Global hiring demand visualization  
✔ Trend analysis across job categories  
✔ Interactive filtering for deeper exploration  
✔ Drill-through analysis for granular insights  

This is not just a dashboard — it’s a **career decision-support system**.

---

## 🛠 Tech Stack & Skills Demonstrated

**Power BI** – Dashboard Development  
**Power Query (ETL)** – Data Cleaning & Transformation  
**DAX (Implicit Measures)** – KPI Development  
**Geospatial Analysis** – Map Visualizations  
**Interactive Reporting** – Slicers, Buttons, Bookmarks, Drill-through  

Core metrics developed:
- Median Yearly Salary  
- Job Count  
- Role-wise Compensation Comparison  

---

## 📊 Analytical Approach

1. Cleaned and standardized raw job posting data.
2. Engineered salary-based and role-based metrics.
3. Designed comparative visualizations (Bar, Line, Area, Map).
4. Built intuitive navigation for executive-level usability.
5. Focused on storytelling through structured dashboard layout.

---

## 🌍 Key Insights Uncovered

- Certain data roles command significantly higher median salaries.
- Hiring demand is geographically concentrated in specific markets.
- Compensation varies substantially across role specializations.
- Trend patterns reveal evolving demand in the data ecosystem.
